# cats

Move your mouse and cat-shaped hearts trail behind it, over a dark background with the word "cats" in the middle.

## Stack

- HTML, CSS, vanilla JavaScript
- Node.js server that serves the static files and proxies random cat images from cataas.com

## Clone

```
git clone https://github.com/Jules0061/cats.git
```

## Enter the folder

```
cd cats
```

## Run

```
npm start
```

Then open your browser at:

```
http://localhost:3000
```

## Files

- index.html main page
- style.css styling and heart mask
- script.js mouse tracking and cat spawning
- server.js Node server and cat image proxy
- package.json start script
